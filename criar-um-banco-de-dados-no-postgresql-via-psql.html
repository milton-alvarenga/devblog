Criar um banco de dados no PostgreSQL via psql

Posted on February 27, 2013 by Alvarenga Milton — Comments

Para efetuar a criação de um novo banco de dados no PostgreSQL, via psql, basta:

CREATE DATABASE <nome_da_base_de_dados> ENCODING 'UTF-8' OWNER <usuario>;  

Caso o novo banco de dados necessite ser uma cópia de um existente, basta utilizá-lo como template na criação, conforme sintaxe abaixo:

CREATE DATABASE <nome_da_base_de_dados> WITH TEMPLATE <nome_da_base_de_dados_a_ser_copiada> ENCODING 'UTF8' OWNER <usuario>;
